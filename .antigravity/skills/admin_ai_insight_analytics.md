# Skill: Admin AI Insight Analytics

**Trigger:** “Add AI insight analytics to admin console”

**Purpose:** Measure the effectiveness of Prophetic Intelligence across all churches and identify opportunities for model improvement.

**Implementation Steps:**

1. Create a dashboard showing:
   - Total AI insights delivered
   - Open rate (pastors viewing insights)
   - Helpfulness ratings (from pastors)
   - Insight categories that perform best
2. Correlate insight effectiveness with church engagement (e.g., churches that act on insights grow faster).
3. Provide a feedback loop: allow admins to review low‑rated insights and either adjust the model or mark them as training data.

**Success Criteria:** Product team can monitor AI performance and continuously improve the PIL Engine.

**Failure Handling:** Show simple counts if rating data is sparse.
